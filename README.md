# Telecom_Churn_Analysis
This is an academic project developed collaboratively by a two-member team (Sneha Adhikari and Aastha Khanal), focusing on analyzing telecom data to understand and evaluate customer churn.
#Summary:
•	Analyze telecom customer data to identify churn patterns and evaluate predictive factors.
•	Performed exploratory data analysis and built models using Python in Jupyter Notebook on a real-world dataset.
•	Concept implemented: Python, Jupyter Notebook, Data Analysis, EDA, Collaborative Development, CSV Data Handling
